The Old Queensland

2019-09-26 1 min read

[July 2017 - November 2017]
Course: Design Computing Studio 1

The Old Queensland is an interactive web application built using the State Library of Queensland. The objective of the website is allowing users to explore historical information through various interesting interactions, visualizing digital records and offering an aesthetic visual representation for users. At the meantime, this project tends to provide a better browsing and learning experience for the target customers of the website.
