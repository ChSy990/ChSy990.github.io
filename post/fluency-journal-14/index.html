RFID Tech Building

I found three different RFID tutorials in Particle's Library and experimented with the instructions. My goal is to complete the connection of One-Photon and RFID Reader this week and successfully receive signals from the RFID Tag. I decide to use one Photon as a hardware prototype which connects with the Mifare RFID-RC522 reader module. Finally, the reader is used to read the UID code of the Mifare Tag. As my concept, the Photon NO.1 use the Particle. Publish function to transfer the RFID data to the database, and Photon No.2 use the Particle. Subscribe function to obtain the database information. Each RFID Tag represents a button on a Korean keyboard, while in Korean, there are two ways of accented and lightly pronounced, so I need to make at least 52 keystrokes. Game Menu

This week I also completed two game's menu design and background image design and provided it to Alan to complete the game's front end and interface design. After this week's work, our Unity game has a complete process. In order to make the game scene of Low Poly more refined, I have reserved much new interface and put the most typical scene in our game. Some game interfaces I do not think it is too complicated, or the functionality on the same interface cannot be too much. So I think that our game can achieve no more than three functions on the same page, too many functions, the player will be particularly annoyed when it comes to this page. I did empathy. If I am a player, it would be enough for me (the player) to get the desired function on one page. However, if it is a sophisticated design with many features, it will make me (the player) need to distract the target function, so I (the player) have to think about what to do and what is the function I want to use. This process means that the game designer has successfully inspired the player's choice of difficulty. These two different game menus and interfaces will be tested after the game is completed, and the final version will be determined based on the user's preferences.
