Anxiety Eliminator Game
2018-12-30 1 min read # Music # Game

[July 2017 - November 2017]
Personal Happy Project

Anxiety Eliminator is an interactive & digital game prototype built by using the Unity and involve some form of physical interaction by using the Leap Motion and MakeyMakey. Players need to match the music and tap the schematic pattern in the game prototype. The player operates the digital game prototype mainly by their gestures activities or by hitting physical objects.
